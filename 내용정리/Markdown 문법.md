# 목록

- 순서가 없는 목록 (-하이픈이나 * 치고 시작)
  - tab하면 소목록 생성
    - 계속 진행
- shift + tab해서 돌아옴
- enter 2번으로 탈출



1. 순서가 있는 목록
   1. tab으로 소목록 생성
      1. 계속 진행
2. 이후는 순서 없는 목록과 같음



# 코드 블록

Inline 코드 블록 `def`

'''python + enter로 파이썬 블럭 생성

```python
# 파이썬 주석
print('hello')
```

'''html + enter로 html 블럭 생성

```html
<!--html 주석-->
<h1>
    hello
</h1>
```

블락에서 탈출해서 enter 치고 싶을 때: **ctrl + enter**
