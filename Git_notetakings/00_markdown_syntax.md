## 목록

- 순서가 없는 목록 (-하이픈이나 * 치고 시작)

    - 계속 진행
- shift + tab해서 돌아옴
- enter 2번으로 탈출

## 코드 블록

Inline 코드 블록 생성

- 아래서 쓰이는 '''는 실제로는 ```(backtick)임



1. '''python + enter로 파이썬 블럭 생성 

```python
# 파이썬 주석
print('hello')
```



2. '''html + enter로 html 블럭 생성

```html
<!--html 주석-->
<h1>
    hello
</h1>
```

- 블락에서 탈출해서 enter 치고 싶을 때: **ctrl + enter**



3. '''bash + enter로 git bash 블럭 생성
